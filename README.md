## Projects

### 🛒 Brazilian E-Commerce Analytics
**Tools:** Python · pandas · matplotlib · seaborn · Power BI · SQL Server

End-to-end analytics project using the Olist Brazilian E-Commerce 
public dataset (~100K orders). Covers a full data pipeline across 
4 Jupyter notebooks, a star schema data model, and a 7-page 
interactive Power BI dashboard.

**Key skills demonstrated:**
- Data cleaning and transformation with pandas
- Exploratory data analysis with matplotlib and seaborn
- Star schema design (4 dims, 2 facts)
- DAX measures and Power BI dashboard development

### 🏙️ NYC Property Sales — SQL Cleaning & Exploration
**Tools:** SQL Server · T-SQL · Excel

Data cleaning and exploratory analysis of New York City property sales 
data using T-SQL. Covers multi-table joins, string manipulation, 
datatype conversion, duplicate detection with window functions, 
and business-driven exploratory queries.

**Key skills demonstrated:**
- Data cleaning with T-SQL (CASE, SUBSTRING, CHARINDEX, CONVERT)
- Duplicate detection with ROW_NUMBER() window function
- Temp tables for staged transformations
- Business queries with GROUP BY, HAVING, RANK()
